全局组件设计 - 智慧零工平台基于 Element Plus，设计以下可复用的全局组件，确保平台整体体验的一致性和开发效率。这些组件将遵循前面定义的视觉风格指南。1. AppHeader.vue (全局顶部导航栏)总体设计: 固定在页面顶部，提供品牌标识、主要导航链接、用户状态及操作入口。背景可采用白色或浅灰色，重要区域可使用毛玻璃效果（如用户下拉菜单）。布局设计 (使用 <el-header> 和 Flexbox/Grid):左侧区域:平台 Logo: SVG 或高质量图片，点击跳转至首页 (/)。平台名称: "智慧零工平台"，使用 H3 或 H4 标题样式，政务红或深灰色。中间区域 (可选，根据页面定):主要导航链接 (<el-menu mode="horizontal">): 如“工作广场”、“技能广场”、“政策指南”。激活状态使用政务红下划线或背景色。全局搜索框 (<el-input placeholder="搜索岗位/技能...">): 在首页或工作广场等页面显示。右侧区域:未登录状态:登录按钮: <el-button type="primary" plain @click="navigateToLogin">登录</el-button> (边框红色，文字红色)。注册按钮: <el-button type="primary" @click="navigateToRegister">注册</el-button> (实心红色)。登录状态:消息通知图标: <el-badge :value="unreadMessageCount" :hidden="unreadMessageCount === 0"><el-icon><Bell /></el-icon></el-badge>，点击跳转至 /user/notifications。用户头像及下拉菜单 (<el-dropdown>):触发器: <el-avatar :src="user.avatarUrl || defaultAvatar" /> 和 <span>{{ user.nickname }}</span>。下拉项 (<el-dropdown-menu>)，可带毛玻璃效果：<el-dropdown-item @click="navigateToUserCenter">个人中心</el-dropdown-item><el-dropdown-item @click="navigateToAccountSettings">账号设置</el-dropdown-item><el-dropdown-item divided @click="handleLogout">退出登录</el-dropdown-item> (退出登录按钮可使用红色文字强调)。按钮设计:登录/注册按钮明确区分主次。下拉菜单中的操作使用文字链接样式。路由跳转:Logo -> /导航链接 -> 各自对应的路由登录按钮 -> /login注册按钮 -> /register消息通知 -> /user/notifications个人中心 -> /user/profile (或用户中心首页)账号设置 -> /user/settings退出登录 -> 执行登出逻辑，然后跳转至 /login 或 /。2. AppFooter.vue (全局底部信息栏)总体设计: 简洁明了，提供版权信息、相关链接等。通常为深灰色或浅灰色背景，白色或浅灰色文字。布局设计 (使用 <el-footer> 和 Flexbox):居中或多列布局。版权信息: © 2025 智慧零工平台 版权所有。相关链接 (<el-link type="info">): “关于我们”、“帮助中心”、“联系我们”、“隐私政策”、“服务条款”。备案号 (如果需要)。按钮设计: 无明显交互按钮，链接使用 Element Plus 的 <el-link> 组件。路由跳转:关于我们 -> /about帮助中心 -> /help(其他链接根据实际页面定义)3. UserCenterSidebar.vue (用户中心侧边导航栏)总体设计: 用于用户中心内部各模块的导航，固定在左侧。可采用毛玻璃背景或与主内容区分的背景色。布局设计 (使用 <el-aside> 和 <el-menu>):顶部显示用户头像和昵称 (可选，如果 UserCenterLayout 已有)。导航菜单 (<el-menu :default-active="activeMenu" router>)：根据用户角色 (零工/雇主) 动态生成菜单项。通用菜单项:<el-menu-item index="/user/profile">个人资料</el-menu-item><el-menu-item index="/user/notifications">消息通知</el-menu-item><el-menu-item index="/user/settings">账号设置</el-menu-item><el-menu-item index="/user/change-password">修改密码</el-menu-item>零工菜单项 (if role === 'freelancer'):<el-menu-item index="/user/freelancer/dashboard">我的工作台</el-menu-item><el-menu-item index="/user/freelancer/applications">我的申请</el-menu-item><el-menu-item index="/user/freelancer/orders">我的订单</el-menu-item><el-menu-item index="/user/freelancer/profile/edit">我的零工档案</el-menu-item><el-menu-item index="/user/freelancer/skills">我的技能</el-menu-item><el-menu-item index="/user/freelancer/wallet">我的钱包</el-menu-item><el-menu-item index="/user/freelancer/reviews">我的评价</el-menu-item><el-menu-item index="/user/freelancer/verification">实名认证</el-menu-item>雇主菜单项 (if role === 'employer'):<el-menu-item index="/user/employer/dashboard">我的工作台</el-menu-item><el-menu-item index="/user/employer/jobs/posted">我发布的工作</el-menu-item><el-menu-item index="/user/employer/jobs/post">发布新工作</el-menu-item><el-menu-item index="/user/employer/orders">我的订单</el-menu-item><el-menu-item index="/user/employer/profile/edit">我的雇主档案</el-menu-item><el-menu-item index="/user/employer/wallet">我的钱包</el-menu-item><el-menu-item index="/user/employer/reviews">我的评价</el-menu-item><el-menu-item index="/user/employer/verification">资质认证</el-menu-item>激活菜单项使用政务红或主蓝高亮。按钮设计: 菜单项本身即为导航按钮。路由跳转: 点击菜单项，通过 <el-menu> 的 router 属性自动跳转到 index 指定的路径。4. PageTitle.vue (页面/模块标题组件)总体设计: 用于在每个页面的主内容区域顶部显示当前页面的标题，也可包含面包屑导航或返回按钮。布局设计:左侧:(可选) 返回按钮: <el-button type="text" :icon="ArrowLeftBold" @click="goBack">返回</el-button>页面主标题: <h1>{{ title }}</h1> (使用政务红或深灰色)。右侧 (可选):面包屑导航 (<el-breadcrumb separator="/">): <el-breadcrumb-item :to="{ path: '/' }">首页</el-breadcrumb-item> ...页面级操作按钮 (如“新增”、“导出”等)。按钮设计:返回按钮: 文字按钮，带左箭头图标。页面级操作按钮: 根据具体功能使用 primary 或 default 类型的按钮。路由跳转:返回按钮 -> router.back()。面包屑 -> 各自对应的路由。5. DataCard.vue (通用数据卡片组件)总体设计: 用于展示工作、订单、申请、技能等列表项信息。白色背景，浅灰色边框，圆角，轻微阴影。可设计多种变体以适应不同内容。Props: title, description, tags (array), status, imageUrl, actions (array of objects for buttons).布局设计 (使用 <el-card>):卡片头部 (header slot): 显示标题，右侧可放置状态标签 (<el-tag>) 或操作按钮。卡片主体:左侧 (可选): 图片 (<el-image :src="imageUrl">)。右侧/主要内容区: 描述、标签列表 (<el-tag v-for="tag in tags">)。卡片底部 (可选): 额外信息或操作按钮组。按钮设计: 卡片内的操作按钮通常使用 type="text" 或 size="small" 的 default 按钮。路由跳转: 卡片本身或卡片内的“查看详情”按钮点击后跳转到对应详情页。6.   }

  }
  .el-dialog__footer {
    // 可选：底部样式调整
  }
}
