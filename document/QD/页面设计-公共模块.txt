页面详细设计 - 公共模块本部分详细描述公共模块下主要页面的总体设计、布局设计、按钮设计及跳转关系。所有设计遵循已定义的视觉风格指南，并使用 Element Plus 组件。1. Home.vue (首页)总体设计: 作为平台门户，信息层级清晰，引导用户高效获取核心信息。顶部采用通栏 Banner 或轮播图，下方为主要功能入口和内容展示区。整体风格大气、专业，点缀政务红。布局设计 (<el-container>, <el-main>):顶部 Banner/轮播区 (<el-carousel> 或自定义组件):内容: 政策宣传、平台活动、重要通知。图片需高清，文字简洁有力。设计: 全宽，高度适中。轮播指示器和箭头使用 Element Plus 默认样式，可定制颜色。快捷入口区 (<el-row :gutter="20"> 和 <el-col>):内容: 主要工作类别（如家政、搬运、派发、客服等）或核心功能入口（如“找工作”、“招零工”、“技能提升”）。设计: 通常为 3-5 个卡片式入口，每个入口包含清晰的图标和文字。卡片可使用 <el-card shadow="hover">，背景白色，图标和文字颜色可使用政务红或主蓝。点击后跳转。按钮: 每个快捷入口本身即为导航按钮。热门/推荐工作区 (<el-tabs v-model="activeJobTab"> 或标题 +列表):内容: “热门推荐”、“最新发布”、“高薪急聘”等标签页，每个标签页下展示 4-8 个工作卡片。设计: 工作卡片使用全局 DataCard.vue 组件。<el-tabs> 激活标签下划线使用政务红。按钮: 工作卡片上的“查看详情”按钮 (type="text", 跳转至 JobDetail)。市场动态/零工风采区 (可选):内容: 简报式展示本地零工市场数据（如图表）、成功案例、优秀零工/企业故事。设计: 使用 <el-card> 分隔模块，图表可使用 ECharts。政策资讯/平台公告区:内容: 列表形式展示最新的政策解读、平台公告。设计: 简洁列表，每项包含标题和发布日期，点击可查看详情。按钮: “查看更多”链接 (type="text", 跳转至政策列表页或公告列表页)。按钮设计与跳转:快捷入口: 点击跳转至对应的工作搜索结果页 (带筛选参数) 或功能页面。例如：“家政服务” -> /jobs?category=家政服务工作卡片“查看详情”: router.push({ name: 'JobDetail', params: { id: job.id } })。“查看更多” (热门工作/政策资讯): 跳转至 /jobs 或 /policy。美术风格:主背景白色。Banner 和快捷入口可使用政务红或主蓝作为点缀色。卡片使用浅阴影，圆角。整体感觉明亮、专业、信息丰富但不拥挤。2. auth/Login.vue (登录页)总体设计: 页面居中布局，简洁明了。主色调采用政务红与白色的搭配，营造专业且醒目的视觉效果。可考虑在背景使用大幅的、模糊的城市剪影或抽象几何图形增加高级感。布局设计 (<el-container> 使内容垂直居中, <el-card class="login-card">):登录卡片 (login-card):宽度: 400px - 450px，居中显示。背景: 白色，可带有轻微的毛玻璃效果的边框或头部。头部:平台 Logo (可选，如果全局 Header 不在此页面显示)。标题: "用户登录" 或 "欢迎登录智慧零工平台" (H2 样式，深灰色)。表单 (<el-form ref="loginForm" :model="loginData" :rules="loginRules">):手机号输入框: <el-form-item prop="phoneNumber"><el-input v-model="loginData.phoneNumber" placeholder="请输入手机号" prefix-icon="User" /></el-form-item>密码输入框: <el-form-item prop="password"><el-input v-model="loginData.password" type="password" placeholder="请输入密码" prefix-icon="Lock" show-password /></el-form-item>操作行 (<el-row justify="space-between">):<el-checkbox v-model="loginData.rememberMe">7天内自动登录</el-checkbox><el-button type="text" @click="navigateToForgotPassword">忘记密码?</el-button> (文字颜色政务红或主蓝)登录按钮: <el-button type="primary" @click="handleLogin" :loading="loading" style="width: 100%; margin-top: 10px;">登 录</el-button> (主按钮，政务红背景，白色文字)。其他登录方式 (可选):分割线: <el-divider>其他登录方式</el-divider>图标按钮: 微信登录、支付宝登录 (使用对应品牌色图标按钮)。注册入口:<span>还没有账号? </span><el-button type="text" @click="navigateToRegister">立即注册</el-button> (文字颜色政务红或主蓝)。按钮设计与跳转:登录按钮:设计: type="primary"，政务红 (#CC0000) 背景，白色文字，尺寸 large。跳转: 成功后根据用户角色跳转至 /user/freelancer/dashboard 或 /user/employer/dashboard 或 / (如果之前有重定向，则跳转至 redirect 路径)。忘记密码按钮:设计: type="text"，文字颜色政务红。跳转: router.push({ name: 'ForgotPassword' })。立即注册按钮:设计: type="text"，文字颜色政务红。跳转: router.push({ name: 'Register' })。第三方登录按钮 (若有): 点击后通常会打开新窗口或重定向到第三方授权页。美术风格:背景可为纯白或浅灰色，登录卡片突出。输入框、按钮遵循 Element Plus 风格，主按钮使用政务红。卡片圆角 8px，可有轻微 box-shadow。3. auth/Register.vue (注册页)总体设计: 与登录页风格保持一致，居中卡片式布局。强调操作的引导性。布局设计 (类似登录页的 <el-card class="register-card">):注册卡片:标题: "用户注册" 或 "创建您的智慧零工账号"。表单 (<el-form ref="registerForm" :model="registerData" :rules="registerRules">):手机号输入框: <el-form-item prop="phoneNumber"><el-input v-model="registerData.phoneNumber" placeholder="请输入手机号" prefix-icon="User" /></el-form-item>验证码输入框:<el-form-item prop="smsCode">
